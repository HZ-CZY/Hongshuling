  - 备份：birds/assets/index-DJNsTAI3.css.dark.bak、index-CBCtj4a_-v2.js.dark.bak。
  - 用 Python 按整 token 精确替换：深蓝面板底 #0a14xx/#0a19xx → 薄荷白 #f6faf3(保留透明度)；浅色文字 #e0f0ff 等 → 深墨绿 #123527/#2a4636/#3c5a4a/#5f7a6a；白色描边 rgba(255,255,255,.x)/#ffffffXX → 深色细描边 rgba(18,53,39,.x)/#123527XX；浅强调文字加深 #50d8d8→#0d9488、#64c8ff→#2f8fd6、#f5c842→#e6a700、#5cdb8d/#7ddf8e→#16a34a。
  - #fff 特殊处理：保留彩色按钮白字(#ff6b6b/#00a381/#008a8a 上的 color:#fff)，其余 color:#fff→#123527。
  - 高德底图 JS 里 amap://styles/dark → amap://styles/fresh(清新明亮)。
- 校验：残留深蓝面板底=0，浅色文字残留=0，白描边残留=0；f6faf3×19、#123527×104、深色描边×46。
- 关键经验：地图暗是因为底图 mapStyle=amap://styles/dark，改 fresh/light 是最大提亮点；配色改写用整 token 正则替换 + 彩色按钮白字占位保护，安全不破坏布局。
