# 全球候鸟迁徙路线资料来源

3D 地图中的路径是用于自然教育的近似可视化，并非实时卫星追踪轨迹。坐标依据公开资料描述的繁殖地、主要迁飞区域和越冬地绘制。

| 物种 | 地图展示路线 | 资料来源 |
|---|---|---|
| 北极燕鸥 | 格陵兰/北极圈 → 大西洋 → 南极海域 | [Cornell Lab - Arctic Tern](https://www.allaboutbirds.org/guide/Arctic_Tern/overview) |
| 斑尾塍鹬 | 阿拉斯加 → 跨太平洋 → 新西兰 | [Cornell Lab - Bar-tailed Godwit](https://www.allaboutbirds.org/guide/Bar-tailed_Godwit/overview) |
| 阿穆尔隼 | 东北亚 → 印度东北部 → 跨印度洋 → 南部非洲 | [Amur falcon](https://en.wikipedia.org/wiki/Amur_falcon) |
| 灰鹱 | 新西兰 → 日本/阿拉斯加/美洲外海 → 新西兰 | [Cornell Lab - Sooty Shearwater](https://www.allaboutbirds.org/guide/Sooty_Shearwater/overview) |
| 家燕 | 欧洲 → 地中海/撒哈拉 → 南部非洲 | [RSPB - Swallow](https://www.rspb.org.uk/birds-and-wildlife/swallow) |
| 美洲鹤 | 加拿大伍德布法罗 → 北美中央走廊 → 得克萨斯湾岸 | [Cornell Lab - Whooping Crane](https://www.allaboutbirds.org/guide/Whooping_Crane/overview) |
| 红喉北蜂鸟 | 加拿大/美国东部 → 墨西哥湾 → 中美洲 | [Cornell Lab - Ruby-throated Hummingbird](https://www.allaboutbirds.org/guide/Ruby-throated_Hummingbird/overview) |
| 穗䳭 | 阿拉斯加 → 加拿大/格陵兰/欧洲 → 东非 | [Cornell Lab - Northern Wheatear](https://www.allaboutbirds.org/guide/Northern_Wheatear/overview) |

## 使用说明

- 距离采用公开科普资料中的常见近似值，个体和年度之间会有差异。
- 路线经过点用于表达迁飞方向，不代表鸟类必然经过地图上的每个精确坐标。
- 原有深圳湾路线继续作为东亚—澳大利西亚迁飞区的本地案例保留。
