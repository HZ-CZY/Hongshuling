# 🕊️ 羽迹·深湾 — 候鸟栖息地地图

深圳湾候鸟迁徙可视化平台。

## 内容
-**'/对照设计稿做了这些对齐/'**
  设计稿特征	调整内容
  面板纯白底	#f6faf3（带绿调）全系 19 处 → #ffffff 纯白（保留各透明度变体）
  深森林绿按钮	主强调色 #0d9488（青绿）→ #166534（深森林绿），渐变按钮同步改为森林绿渐变
  青色品牌残留清理	原 #00c8c8 从 39 处清到仅 7 处装饰性用法（地图标记点/发光/加载动画），UI 状态色全部统一到森林绿系
  中性灰文字	绿灰色 #3c5a4a / #5f7a6a → 中性板岩灰 #4a5568 / #64748b，白底上更干净
  轻投影	重暗影 0 8px 32px rgba(0,0,0,.5) → 轻柔阴影 0 4px 24px rgba(0,0,0,.08)，匹配极简风
  最终色调分布：纯白面板 + 深森林绿强调 + 深墨绿文字 + 中性灰辅助 + 青色仅用于地图标记点缀
- **`/birds/`** — 2D 生态地图（Vue3 + 高德地图 AMap）
  - 候鸟物种信息、迁徙路线、观测数据、气象站
- **`/birds/space.html`** — 3D 深空迁徙视图（Three.js）
  - 立体地球、6条迁徙路线动画

## 技术栈

- 2D: Vue 3 + Vite + 高德地图 JS API 2.0
- 3D: Three.js + CSS2DRenderer
- 部署: nginx 静态托管
